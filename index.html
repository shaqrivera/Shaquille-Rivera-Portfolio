Pro Event Planning

Pro Event Planning is an app that allows users RSVP to events, and get weather data for that event all on the same page. Once daily, Pro Event Planning sends an email to all users that have made an RSVP for an event, and emails them updated info about the weather on the day of the event.

Repository Live Deployment
